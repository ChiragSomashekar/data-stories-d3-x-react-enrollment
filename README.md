# D3 × React : Data Stories

A simple data visualization built with D3 and React to show where data storytellers are building around the world.

---

## Overview

This project displays a horizontal bar chart of countries and the number of data storytellers enrolled.

Focus:
- clean design  
- simple interaction  
- easy comparison  

---

## Tech Stack

- React  
- D3.js  
- Vite  
- gh-pages  

---

## Features

- Hover to see values  
- Minimal layout  
- SVG-based chart
